How to Use

01

Get Facebook Appstate

You need to obtain your Facebook appstate (cookies) to login. You can use tools like:

  • Facebook Cookie Extractor
  • Browser DevTools
  • Third-party cookie tools
02

Paste Appstate

Copy your entire appstate JSON and paste it into the text area on the homepage. Make sure it's valid JSON format.

03

Configure Bot Settings

Choose which commands to enable:

  • ping - Check bot response time
  • help - Show help information
  • info - Bot statistics
  • echo - Repeat messages
  • ai - AI chatbot
  • tiktok - TikTok downloader

Set a command prefix (default is /)

04

Add Admin (Optional)

Enter your Facebook User ID if you want admin privileges. Otherwise, leave it blank.

05

Login Bot

Check the terms checkbox and click "Login Bot". Your bot will connect to Facebook Messenger!
